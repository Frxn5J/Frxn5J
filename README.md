## 📌 &nbsp;Featured Projects

### 🚀 [VagaRoute AI](https://github.com/Frxn5J/vagaroute-ai)

Self-hosted AI gateway with a single OpenAI-compatible API endpoint for multiple LLM providers.  
It focuses on free-tier models first and includes automatic failover, rate limiting, caching, cost tracking and a web dashboard.

**Tech highlights:** TypeScript, Bun, SQLite, AI APIs, OpenAI-compatible API, Docker.

---

### 🎬 [TeleStremio AutoRenameBot](https://github.com/Frxn5J/TeleStremio-autorenamebot)

Telegram bot for renaming and sending video files to a channel without storing them locally.  
It supports movie/series workflows, detects quality, uses Telegram `file_id`, and can integrate TMDB and LLM providers for smarter metadata extraction.

**Tech highlights:** Python, Telegram Bot, Docker, SQLite, TMDB API, LLM API integration.

---

### 📰 [NDO Noticias Front](https://github.com/Frxn5J/ndo-noticias-front)

Aplicación móvil en React para el noticiero <b>Noticias De Ojuelo</b>.  
Proyecto desarrollado con frontend en React, backend en Laravel y base de datos PostgreSQL.

**Tech highlights:** React, Laravel, PostgreSQL, mobile-first development.
